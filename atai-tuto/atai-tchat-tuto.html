Description

Le code propose une interface Web classique d'affichage de messages type "mail" pour le tchat de l'application ATAI. Les messages que s'envoient les membres du fow via l'application peuvent tous prendre cette forme, avec l'avatar du destinataire dans la barre de navigation et de l'expéditeur dans l'entête du message, afin de simuler correctement l'affichage du message. Il est possible mais facultatif d'intégrer directement des images dans le corps du message, ou bien sous forme de pièces-jointes avec option thumbnail pour les images (simulées dans cette demo) ou option fichiers attachés (idem avec tous les formats possibles). Voici un descriptif exhaustif de l'ensemble :

  • La barre de navigation est composée du logo (version inline) de l'app, du nom de la messagerie, et de l'avatar et du nom du destinataire du message. L'idée est que, si X envoie un message à Y sur le tchat, on ait accès à la version du message telle que reçu par le destinaire, soit Y. X étant l'expéditeur, ses informations apparaissent dans l'entête du message lui-même, exactement comme sur les messageries Web standards. Nota bene : le logo et le titre + icône de la messagerie sont injectés directement par le code CSS afin de ne pas surcharger le HTML.
  • L'entête du message contients les éléments personnalisables à compléter suivants : l'avatar de l'expéditeur, le nom complet de l'expéditeur, le lien vers le profil ATAI de l'expéditeur sur le fow, le lien vers le profil ATAI du destinataire sur le fow, la date d'envoi du message et l'icône (facultative) de pièce-jointe.
  • Le corps du message, dans la partie rose, contient le titre/objet du message et le contenu du message. Le titre et les paragraphes sont à compléter. Chaque paragraphe doit être inclus entre balises <p></p>.
  • Le pied du message est facultatif : on peut intégrer des thumbnails d'images partagées par le personnage, ou des fichiers attachés avec le format personnalisable. On ajoute l'icône de pièce-jointe dans l'entête si le pied contient des pièces-jointes.

Code

Voici le code HTML à copier-coller puis à compléter pour créer un message de l'app ATAI d'un personnage : https://raw.githubusercontent.com/manonstv/JM-LS/refs/heads/main/atai-tchat/atai-tchat.html.

Vous accéderez au code "brut" de la partie HTML, qu'il suffit de copier-coller entre balises [code][/code] sur le forum pour le proposer aux membres.

Consignes

  • Les parties du code à compléter sont signalées par un commentaire en début de ligne. Pour les URL d'images et les liens à compléter, les indications sont signalées en lettres capitales et _underscores_ dans les balises d'images et de liens concernées.
  • Ne pas ajouter d'espaces ou de sauts de lignes. Ils sont inclus et gérés par défaut par le code CSS.
  • Tout ajout doit se faire par strict retour à la ligne en intégrant les parties de codes fournies par ce tutoriel.
  • Ne supprimer aucun caractère ou aucun bout de code à l'exception de ceux expliqués dans ce tuto, car ils ont tous une utilité.
  • Encore une fois, la partie HTML ne doit contenir aucun saut de ligne. Il faut préciser aux membres que le code ne contient AUCUNE erreur. Ils ne doivent donc pas retirer le moindre caractère du code, y compris les commentaires HTML cassés, sous peine de compromettre l'intégrité du rendu.

Astuce : les injections automatiques de sauts de lignes (<br>) intempestifs par le script de forumactif sont contournées grâce à une astuce donnée par Yuna : afin de rendre le code HTML + lisible, les retours à la ligne sont simulés par le "cassage" de commentaires HTML (<!-- -->), permettant des retours à la ligne dans le code source afin de le rendre moins dense et plus facilement personnalisable, le tout sans conséquence dans le rendu.

Personnalisation

  • Pour ajouter ou supprimer un paragraphe dans le corps du message, il suffit d'ajouter ou de supprimer l'une de ces lignes de code :
Paragraphe --><p>Ici</p><!--
  • Pour ajouter des thumbnails d'images partagées par le personnage, il faut ajouter ce bloc de code (ou le supprimer du code par défaut si le membre n'en veut pas) :
--><!-- DEBUT DU CODE : IMAGES EN PIECES JOINTES (FACULTATIF) --><!--
--><div class="img-list"><!-- 
--><img src="URL_IMAGE_THUMBNAIL" alt="..."><!-- 
--><img src="URL_IMAGE_THUMBNAIL" alt="..."><!-- 
--><img src="URL_IMAGE_THUMBNAIL" alt="..."><!-- 
--></div><!-- 
--><!-- FIN DU CODE : IMAGES EN PIECES JOINTES (FACULTATIF) --><!--
  • Pour ajouter ou supprimer des images de la partie thumbnails, il suffit d'ajouter ou de supprimer l'une de ces lignes de code :
--><img src="URL_IMAGE_THUMBNAIL" alt="..."><!--
  • Pour ajouter des fichiers comme pièces-jointes partagées par le personnage, il faut ajouter ce bloc de code (ou le supprimer du code par défaut si le membre n'en veut pas) :
--><!-- DEBUT DU CODE : FICHIERS EN PIECES JOINTES (FACULTATIF) --><!--
--><div class="pj-list"><!--
Fichier .pdf --><p class="pj-item pj-pdf">Document.pdf</p><!--
Fichier .doc --><p class="pj-item pj-doc">Document.doc</p><!--
Fichier CV --><p class="pj-item pj-cv">Document.pdf</p><!--
Fichier code --><p class="pj-item pj-code">Document.html</p><!--
Fichier jpg --><p class="pj-item pj-jpg">Document.jpg</p><!--
Fichier png --><p class="pj-item pj-png">Document.png</p><!--
Fichier mp4 --><p class="pj-item pj-mp4">Document.mp4</p><!--
Fichier mp3 --><p class="pj-item pj-mp3">Document.mp3</p><!--
--></div><!--
--><!-- FIN DU CODE : FICHIERS EN PIECES JOINTES (FACULTATIF) --><!--
  • Pour ajouter ou supprimer des fichiers de la partie pièces-jointes, il suffit d'ajouter ou de supprimer l'une de ces lignes de code (et de préciser le format de fichier dans la classe et dans le nom donné au fichier) :
Fichier .pdf --><p class="pj-item pj-pdf">Document.pdf</p><!--
  • Les classes disponibles pour 8 types de fichiers possibles sont les suivantes (elles génèrent automatiquement les icônes correspondantes) : pj-pdf, pj-doc, pj-cv, pj-code, pj-jpg, pj-png, pj-mp4, pj-mp3.
  • Si le message ne contient aucune pièce-jointe partagée par le personnage, il faut retirer l'icone correspondante dans l'entête du message. Pour ce faire, il faut supprimer la classe have-pj de cette partie du code :
Date d'envoi et icon pièce-jointe --><p class="mail-date have-pj">Ici</p></div></div><div class="mail-body"><!--

Résumé

Voici en définitive la liste exhaustive des parties du code à compléter (dans l'ordre) :

  • Barre de navigation : URL vers l'avatar ATAI du destinataire et nom complet du destinataire.
  • Entête du message : URL vers l'avatar ATAI de l'expéditeur, nom complet de l'expéditeur, lien vers le profil ATAI de l'expéditeur, identifiant ATAI de l'expéditeur, lien vers le profil ATAI du destinataire, identifiant ATAI du destinataire, date d'envoi du message, ajout/suppression de la classe have-pj pour l'icône de pièce-jointe.
  • Corps du message : objet/titre du message, paragraphes du message.
  • Pied du message (facultatif) : bloc de code pour les images en thumbnail + URL des images, bloc de code pour les fichiers en pièces-jointes + modifier les classes associées au format du fichier choisi.
